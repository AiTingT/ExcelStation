# Excel 智能助手

一个**本地运行**的 Excel 处理工具，支持超大文件浏览、数据清洗、透视表、多文件对比和 AI 对话查询。所有数据保存在你电脑上，不会上传到云端。

---

## 🚀 三种使用方式，任选其一

### 方式 A：下载即用（推荐给普通用户）

如果你已经拿到 `ExcelStation` 可执行文件：

1. **双击** `ExcelStation` 运行（Windows 是 `ExcelStation.exe`）
2. 浏览器会自动打开，直接开始用

> 首次启动可能需要 10 秒左右，请耐心等待。

---

### 方式 B：从源码运行（需要 Python）

适合有 Python 环境，或想自己从源码启动：

1. 安装 [Python 3.10+](https://www.python.org/downloads/)
2. **macOS / Linux**：双击 `start.sh`，或在终端运行 `./start.sh`
3. **Windows**：双击 `start.bat`

脚本会自动创建虚拟环境、安装依赖、启动服务并打开浏览器。

---

### 方式 C：打包成单文件（自己打包）

适合需要分发给同事的场景：

1. 安装 Python 3.10+
2. **macOS / Linux**：运行 `./build.sh`
3. **Windows**：运行 `build.bat`

打包完成后在 `dist/` 目录生成可执行文件，**对方无需安装 Python，直接双击就能用**。

## ✨ 功能一览

| 功能 | 说明 |
|------|------|
| 📊 数据浏览 | 上传 Excel/CSV 后快速浏览，支持筛选、排序、**双击单元格编辑** |
| 📈 透视表 | 行/列/值字段自由配置，多种聚合方式 |
| 🧹 数据清洗 | 去重、填充空值、正则替换、列拆分、类型转换（**操作前可预览**） |
| 🔍 对比分析 | 两个 Excel 文件按指定列对比差异，结果分页查看 |
| 🤖 AI 对话 | 用自然语言查询数据，例如「销量前 10 的城市」 |
| 🛠️ 工具箱 | 编码转换、时间戳、JSON 格式化等 8 个常用小工具 |
| ⮰ 撤销 | 所有清洗操作、单元格编辑都支持一键撤销 |

> 💡 虚拟滚动：打开几十万行的大文件也很流畅，不会卡顿。

## 🤖 配置 AI（可选）

如需使用 AI 对话功能，点击页面右上角「设置」：

| 配置项 | 说明 |
|------|------|
| 服务商 | 支持 DeepSeek / OpenAI / 通义千问 / Ollama（本地） |
| API Key | 在服务商官网申请 |
| 模型名 | 例如 `deepseek-chat`、`gpt-4o-mini`、`qwen-plus` |

> 没有 API Key？AI 对话以外的其他功能照常可用。

## 💾 数据存储

- 所有数据保存在你电脑本地（程序所在目录的 `data/` 文件夹）
- **不会上传到任何服务器**
- 使用 SQLite 数据库，单文件易备份

如需使用 MySQL（多人共享场景）：在页面右上角「数据库」里配置连接信息。

## ❓ 常见问题

**Q：启动后黑窗口一闪而过？**
A：Windows 用户请**右键以管理员身份运行** `start.bat`，或查看命令行报错信息。

**Q：浏览器没自动打开？**
A：手动访问 http://127.0.0.1:8000 。

**Q：提示"端口被占用"？**
A：关掉其他占用 8000 端口的程序，或设置环境变量 `PORT=其他端口` 后启动。

**Q：大文件打开慢？**
A：首次解析需要几秒到几十秒（取决于文件大小），之后再查看就秒开。

**Q：想分发给同事？**
A：运行 `./build.sh`（Mac）或 `build.bat`（Windows）打包成单文件，对方无需装 Python。
